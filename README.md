# Финальный проект по автоматизации тестирования VK

## Описание проекта
Автоматизированные тесты для проверки функциональности VK (vk.com), включая UI и API тесты.

## Структура проекта
- `configs/` - конфигурационные файлы
  - `env.py` - настройки окружения
  - `test_data.py` - тестовые данные
- `pages/` - buttons_links_page классы для UI тестов
- `tests/` - директория с тестами
  - `test_ui.py` - UI тесты
  - `test_api.py` - API тесты
- `utils/` - вспомогательные функции
